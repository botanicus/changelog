h1. About

Simple CHANGELOG parser for Ruby.

h1. Usage

<pre>
changelog = CHANGELOG.new("/path/to/my/changelog")
puts "Changes in the last version #{changelog.last_version_name}: \n-  #{changelog.last_version_changes.join("\n-  ")}"
</pre>
